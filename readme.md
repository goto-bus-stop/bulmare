# bulmare

Thin functional React components for [Bulma][].

## Installation

```shell
npm install bulma bulmare
```

## Usage

```js
import { Notification, Delete } from 'bulmare'

<Notification danger>
  <p>Something bad happened!</p>
  <Delete />
</Notification>
```
